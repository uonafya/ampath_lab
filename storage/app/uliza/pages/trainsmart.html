TrainSMART is an acronym standing for Training System Monitoring and Reporting Tool. It was first built and put to use in 2008 and is currently undergoing an overhaul to improve on its functionality.

The application is built using open source software tools which include Linux, Apache, MySQL and PHP. It is accessible to users over the internet and employs user authorization, authentication and is role based allowing different user groups access to various functionalities.
